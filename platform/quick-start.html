Platform ​

¿Qué es Broxel Platform? ​

Broxel Platform es una herramienta CLI diseñada para acelerar el desarrollo de proyectos .NET al ofrecer configuraciones preestablecidas que permiten a los desarrolladores iniciar rápidamente con estructuras y patrones de código recomendados. Simplifica la creación de nuevos proyectos, asegurando consistencia y eficiencia desde el inicio del desarrollo.

Comandos de la CLI ​

De forma predeterminada, se instalan los siguientes comandos:

Comandos básicos ​

  • create
  • configure

¿Cómo instalar Broxel Platform? ​

Prerequisitos ​

  • .NET SDK versión 8.
  • Terminal para acceder a Broxel Platform a través de su command line interface (CLI).
  • Visual Studio 2022 >= v.17.10.
  • Configuración de Nuget Sources asociando el Feed:
bash
https://pkgs.dev.azure.com/BroxelExt/Globaltask/_packaging/GeneralPackages/nuget/v3/index.json

Instalación ​

Utiliza el siguiente .NET Core CLI comando desde la terminal de tu sistema operativo para instalar Broxel Platform.

bash
dotnet tool install -g Platform
powershell
Install-Package Platform -g
